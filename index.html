Introduction

JavaScript is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions.

It is most well-known as the scripting language for Web pages, but it is also used in many non-browser environments such as Node.js.

JavaScript is a prototype-based, multi-paradigm, single-threaded, dynamic language, supporting object-oriented, imperative, and declarative programming styles.

console.log("Hello, World!");
  • JavaScript runs in the browser
  • JavaScript can manipulate the DOM
  • JavaScript supports asynchronous programming
Variables and Data Types

JavaScript has three ways to declare variables: var, let, and const.

The let keyword declares a block-scoped variable, optionally initializing it to a value.

The const keyword creates a constant reference to a value. Constants cannot be reassigned.

let name = "Alice"; const age = 30; var isStudent = true; typeof "hello"; // returns string typeof 42; // returns number typeof true; // returns boolean
  • String: text values enclosed in quotes
  • Number: integer and floating point numbers
Functions

A function in JavaScript is a block of code designed to perform a particular task.

Functions are executed when they are called. You can pass data into a function as parameters.

Arrow functions provide a shorter syntax for writing function expressions and do not have their own this context.

function greet(name) { return "Hello, " + name + "!"; } console.log(greet("Alice"));
  • Functions can return values using the return statement
  • Arrow functions use the => syntax for concise function definitions
Control Flow

Control flow statements allow you to control the order in which code executes based on conditions.

The if statement executes a block of code if a specified condition is true.

The for loop repeats a block of code a number of times based on a condition.

if (age >= 18) { console.log("Adult"); } else { console.log("Minor"); }
  • if/else statements handle conditional logic
  • for and while loops handle repetition
  • switch statements handle multiple conditions efficiently
Objects and Arrays

Objects in JavaScript are collections of key-value pairs used to store related data.

Arrays are ordered lists of values and can contain any type of data including other arrays.

Both objects and arrays are fundamental data structures in JavaScript programming.

const person = { name: "Alice", age: 30, city: "New York" }; console.log(person.name);
  • Objects store data as named properties
  • Arrays use numeric indices starting at zero
